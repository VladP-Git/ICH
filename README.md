# ICH



\# Фильмотека Sakila — Поисковая Система



Учебный проект по разработке приложения для поиска информации о фильмах с использованием обезличенных данных. Проект объединяет фронтенд-технологии и гибридную бэкенд-архитектуру на базе реляционной (SQL) и документоориентированной (NoSQL) баз данных.



\## Цель проекта

Создание локального (возможно с последующим переносом на VPS) веб-интерфейса, заменяющего стандартный консольный ввод, для удобного поиска кинокартин, фильтрации по жанрам/годам и логирования пользовательской активности.



\## Планируемая архитектура и структура проекта



```text

movie\_search\_project/

│

├── backend/

│   ├── app.py                  # Веб-сервер и роутинг (Flask / FastAPI)

│   ├── mysql\_connector.py      # Модуль интеграции с реляционной БД (Sakila)

│   ├── log\_writer.py           # Модуль записи логов поиска в NoSQL MongoDB

│   └── log\_stats.py            # Аналитический модуль (Топ-5 запросов)

│

├── templates/                  # Интерфейс (HTML-страницы)

│   ├── index.html              # Главная страница: поиск, фильтры, сетка фильмов

│   └── stats.html              # Страница аналитики и статистики запросов

│

├── static/                     # Статические ресурсы веб-сайта - стили, JS и картинки

│   ├── css/

│   │   └── style.css           # Кастомные стили поверх Bootstrap

│   ├── js/

│   │   └── main.js             # Скрипты для динамической пагинации (по 10 фильмов)

│   └── images/                 # Локальные медиа-ресурсы

│

├── requirements.txt            # Список внешних зависимостей Python

└── README.md                   # Описание проекта

```



\## Стек технологий

\*   \*\*Frontend:\*\* HTML5, CSS3, Bootstrap 5 (адаптивная сетка, компоненты карточек).

\*   \*\*Backend:\*\* Python 3.x (Flask/FastAPI), SQL-запросы с использованием JOIN.

\*   \*\*Реляционная БД:\*\* MySQL (учебная база данных Sakila), администрирование при помощи MySQL Workbench / DBeaver 

\*   \*\*NoSQL БД (Логирование):\*\* MongoDB, администрирование при помощи MongoDB Compass.

\*   \*\*Управление версиями:\*\* Git / GitHub.
